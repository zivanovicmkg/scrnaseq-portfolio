# Project 09 — Cell-Cell Communication Analysis

**Tool:** CellChat (R) — possibly compared to CellPhoneDB (Python)
**Dataset:** Re-uses TME data from Project 02 (Wu 2021 breast cancer)
**Reference:** Jin S, Guerrero-Juarez CF, Zhang L, et al. *Inference and analysis of cell-cell communication using CellChat.* Nat Commun 12, 1088 (2021). DOI: 10.1038/s41467-021-21246-9

## Why this project

Advanced downstream analysis — ligand-receptor signaling across cell types. Heavily requested in TME, immune, and developmental projects. Demonstrates beyond-clustering analytical capability.

## Goals
1. Compute signaling networks between TME cell types
2. Identify dominant ligand-receptor pairs (e.g., CXCL/CXCR, TGF-β signaling)
3. Visualize circle / chord / heatmap plots of intercellular signaling
4. Compare two conditions if metadata permits (e.g., subtype-stratified signaling)

## Estimated effort
**8–10 hours.**
