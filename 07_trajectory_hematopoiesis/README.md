# Project 07 — Trajectory Inference: Hematopoiesis

**Tool:** Palantir + Scanpy (Python)
**Dataset:** Setty et al. 2019 — human bone marrow CD34+ HSPCs
**Reference:** Setty M, Kiseliovas V, Levine J, et al. *Characterization of cell fate probabilities in single-cell data with Palantir.* Nat Biotechnol 37, 451–460 (2019). DOI: 10.1038/s41587-019-0068-4

## Why this project

Second methodological showcase, complementary to #06. Trajectory + fate probability — different conceptual framework from velocity (pseudotime + branching probabilities). Demonstrates breadth in dynamic scRNA-seq analysis.

## Goals
1. Reproduce Palantir's branching trajectory for HSPC → erythroid / lymphoid / myeloid fates
2. Compute fate probabilities per cell
3. Plot canonical pseudotime + fate-probability heatmap
4. Identify gene trends along each lineage

## Estimated effort
**10–12 hours.**
