# Project 10 — Cross-Platform Re-Analysis & Benchmarking

**Tool:** Scanpy + Seurat (parallel, same dataset)
**Dataset:** TBD — pick a well-cited published scRNA-seq study with conclusions that can be tested

## Why this project

The capstone. Shows critical thinking — not just "run the pipeline," but "verify the conclusion." Two independent analyses on the same data with different tools, compared transparently.

## Goals
1. Reproduce the published clustering / annotation in both Scanpy and Seurat
2. Compare: cluster membership agreement, marker gene overlap, downstream conclusions
3. Document any disagreements and discuss methodological reasons
4. Provide a side-by-side report

## Candidate datasets
- Tirosh et al. 2016 (melanoma, GSE72056) — small, fast, well-cited
- Tabula Sapiens subset — clean, well-annotated
- Any of the earlier portfolio projects re-done in the alternative tool

## Estimated effort
**15–20 hours.** Most ambitious; do last.
